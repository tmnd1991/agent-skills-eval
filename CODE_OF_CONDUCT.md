# Code Of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Expected Behavior

- Be respectful and direct.
- Assume good intent while keeping technical feedback specific.
- Focus criticism on code, docs, tests, and behavior.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.

## Enforcement

Maintainers may remove comments, close issues, or block participants who violate this code of conduct.
